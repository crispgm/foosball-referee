## Introduction

A web application for foosball referee.

### Features

- Quick access via a single page web application.
- Designed for Portrait Mode of mobile phone.

## Dev

```shell
# Install dependencies
npm install

# Lint
npm run lint

# Serve with static server
npm run serve
```
